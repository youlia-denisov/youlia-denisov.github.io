Streamlit App

WattWise — Electricity Dashboard

Multi-user Streamlit dashboard for analysing household electricity consumption from Israeli IEC exports. Runs a full pipeline: preprocessing, K-means clustering, four outlier-detection methods, weather correlation via Open-Meteo, and a discount plan comparison engine. Every browser session is fully isolated — no data leaks between users.

⚠️ Demo mode: Several features require a real IEC export file — upload yours to unlock them.
